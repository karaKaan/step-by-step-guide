# Step-by-Step Guide

The step-by-step Guide shows a list of Foods and Drinks with there Ingredients and Direction.
After the refreshing of the Webbrowser the States will remain. 
It shows one by one how the finish the Goodie.


## Framework and Packages Used
* React.js
* React-Router-Dom
* Express.js
* Nodemon

## Guide to start the Application
1. In a Folder of your choice run this Command in a Terminal. <br/>
`git clone https://github.com/karaKaan/step-by-step-guide.git`
2. Redirect the terminal into the step-by-step-guide Folder.
3. Install the following: <br/>
`npm install create-react-app`
4. Open your Code Editor using `code .` in your Terminal.
5. Go to src/server and start the integrated Terminal.
6. Run the command `node express.js`
7. In a new Terminal, at the root, run the command `npm start`
